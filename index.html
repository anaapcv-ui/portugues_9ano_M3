2
O sistema de código

🔐 Como funciona?

Quando o aluno termina o Quiz ou Simulado, ele recebe uma nota e um código de liberação. Esse código serve para desbloquear o gabarito com as explicações detalhadas.

Os códigos têm este formato:

Quiz: QUI-8-FK (8 = acertos)
Simulado: SIM-15-AB (15 = acertos)

Por que o código existe? Para que o aluno tente responder sozinho antes de ver as respostas. O ideal é que o adulto só libere o gabarito depois que o aluno terminar e ver sua nota.

3
Dicas para mãe/pai
📱 Todos os materiais funcionam no celular e no computador. Basta abrir o link — não precisa instalar nada.
📖 A ordem sugerida é: Resumo (para revisar) → Leitura Guiada (para aprofundar) → Quiz (para testar) → Simulado (para simular a prova). Mas cada material funciona sozinho!
🔑 Quando seu filho terminar o Quiz ou Simulado, ele vai mostrar o código. Anote o código e digite no campo de liberação para abrir o gabarito. Assim ele pode estudar os erros.
🎯 No Simulado, a meta é 70% (14 acertos em 20). Se não atingir, revise o Resumo e tente novamente — as questões são embaralhadas toda vez!
✏️ Na Leitura Guiada, o aluno responde com suas próprias palavras antes de ver a resposta modelo. Incentive a escrever sem consultar — o esforço de pensar sozinho é o que faz fixar!
⚠️ O conteúdo da prova inclui: ambiguidade lexical e fonética, os 6 processos de derivação e os afixos (prefixos e sufixos). A pegadinha mais cobrada é a diferença entre derivação prefixal e sufixal × parassintética!
Bons estudos! 💙 Cada material funciona de forma independente — use quando precisar.
